How to Help

Truckerswelcome.ca is an open source initiative to help our truck, delivery and transit drivers who are putting themselves at risk to keep our supply chain moving during the COVID-19 pandemic and finding it increasingly difficult to get access to the basic services they need due to social distancing measures.

For more information on the challenges they are facing see this recent CBC news article.

Developers

If you are a developer, here's a list of next steps to get this off the ground:
  • Set up git repository for the project
  • Set up a slack group for the project
  • Set up the DevOps infrastructure so we can deliver stable functional increments
  • Implement the application database (list of businesses, their location and services they are offering to drivers)
  • Front-end development for the application interface (browser based but primarily usable on mobile devices)
  • Backend development to support location based searches and other server-side functions
  • Security leads to ensure security best practices are followed at the database, source code and operational level
The github project is here: https://github.com/truckerswelcome/webapp
Use this link to join the slack channel: https://join.slack.com/t/truckerswelcome/shared_invite/zt-dakd3cn7-ndNJDH0JyUBTy09ABT0A1Q

Community

This initiative will only succeed with support from the broader community. Here are a few things you can do to help:
  • Spread the word. We need data for this to be useful to our drivers. The more businesses in the database, the more helpful this will be to drivers. It would be great to get help from the media to inform businesses and drivers once the minimum viable product is up and running.
  • Gather & input data. Do you know of a business that is offering services to drivers? Send that information over or use the application interface (once it is operational) to enter the information.
  • Help with resources. The application is running in Google Cloud and using the Google Maps API. If you have contacts within Google that can help with cloud resources please reach out to them.

Interested in helping? Send an email to help@truckerswelcome.ca
